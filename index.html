Sobre

Atualmente estou no cargo de desenvolvedor web front end pela TruckPad utilizando a biblioteca React.js.

Sou um desenvolvedor web na área de front end. Possuo anos de experiência com o framework Vue.js e com o framework Nuxt.js. No desenvolvimento dos meus projetos eu prezo pela responsividade e pela acessibilidade, ou seja, projetos capazes de serem acessados em qualquer dispositivo, com qualquer tamanho de tela e por usuários com as suas respectivas limitações visuais ou motoras.

Minha lista de conhecimentos com maior experiência:

  • JavaScript e TypeScript
  • React.js
  • Vue.js
  • Nuxt.js
  • Sass
  • Pug
  • Git

Estou sempre buscando me atualizar nas tecnologias que utilizo como também sempre estou procurando novos conhecimentos. Atuamente estou estudando e aprendendo React.js, React Native e Node.js através da plataforma Rocketseat.
